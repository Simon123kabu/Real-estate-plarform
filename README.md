# Real Estate Property Listing Platform

A centralized, web-based property listing platform designed to connect real estate agents with buyers and renters. Built to solve the problem of outdated, 
duplicated, and unverified listings common on social media and classified sites, this platform provides a seamless search experience for property seekers and 
professional management tools for independent agents.

## 🚀 Key Features

* **Role-Based Access Control:** Secure JWT authentication tailored for  Agents, and Buyers.
* **Comprehensive Property Management:** Agents can create, edit, and manage listings with support for multiple image uploads.
* **Advanced Search & Filtering:** Buyers can easily discover properties by filtering location, price, property type, and number of bedrooms.
* **Favorites System:** Seekers can save and track their preferred property listings.
* **Responsive Design:** Optimized performance and layout for both desktop and mobile browsing.
